正向代理

Posted on 2022-03-12 Edited on 2023-12-06

正向代理通常用在远程访问某个环境中的。常见的正向代理工具包括squid、nginx、3proxy。

squid

老牌的正向代理工具。

安装：yum install squid && systemctl start squid

squid默认会监听在3128端口号。

缺点：如果修改了本地的/etc/hosts文件，则需要重启squid后才可以更新。
